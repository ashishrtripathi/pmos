## Status: backlog
## Priority: critical
## Points: 13
## Created: 2026-07-22
## Project: voxstyle

### User Story
As a content creator, I want to enter a subject and target length, then see an editable scene table generated by AI so that I can review and approve the script before any processing begins.

### Acceptance Criteria
- [ ] Given a subject and length, when I submit the form, then Claude generates a scene table
- [ ] Given the scene table, when I view it, then all columns from spec §3 are present
- [ ] Given the scene table, when I edit a row, then the change is saved locally
- [ ] Given the scene table, when I click "Approve", then the pipeline proceeds
- [ ] Given the scene table, when I click "Approve", then editing a row flags it as stale
- [ ] Given chart data, when the LLM writes a metric, then structured JSON is emitted

### Business Rules
- Scene table is an editable UI grid (not just text)
- Approval gate: no processing starts until user approves
- Editing after approval flags that row as stale
- Chart data must be structured JSON, not invented downstream
