## Implemented Features

### 1. Video Generation Pipeline
- **Complete end-to-end pipeline** from subject → rendered MP4
- **20-scene hardcoded demo** (AI History topic) in `Root.tsx`
- **Express API** with `/generate-video`, `/video-status`, `/video-result`, `/health`
- **Job queue** with UUID-based tracking (in-memory)

### 2. Script Generation
- **Gemini API integration** (primary) for script + scene table generation
- **Anthropic Claude** as fallback
- **Scene structure**: 20 scenes with voiceover, prompts, chart data, interactions
- **Timepoints** generated for audio synchronization

### 4. Image Processing (Python)
- **Background removal**: `rembg` with U²-Net (ONNX runtime)
- **Halftone generation**: Custom Pillow-based dot pattern
- **Caching**: MD5 hash of input + config prevents reprocessing
- **Output**: Transparent PNG with halftone dots

### 5. Voiceover & Audio
- **Google Cloud TTS** (primary) with SSML time marking
- **Edge TTS** (free fallback) via Python script
- **Timepoint generation**: Estimated (word-count based) + placeholder for Whisper alignment
- **Per-scene audio files**: `/audio/scene_N.mp3`

### 6. Remotion Video Composition
- **MainVideo**: Orchestrates all scenes in sequence
- **Scene component**: Foreground + Midground + Background layers
- **Transforms**: Position, scale, rotation, opacity, z-index per scene
- **Interaction types**: `parallel` (parallax), `facing` (contrast)
- **Chart support**: Bar, pie chart overlays in scenes

### 7. Rendering
- **Remotion Studio**: Development UI at `:3000`
- **FFmpeg rendering**: Production MP4 output to `out/video.mp4`
- **Duration**: 3895 frames @ 30fps = ~130 seconds
- **Resolution**: 1920×1080

### 8. Configuration & Environment
- **TypeScript strict mode** with path aliases (`@/*`)
- **Environment validation** with required/optional key categorization
- **Hot reload** via Remotion Studio
