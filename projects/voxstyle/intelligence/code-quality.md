## Recommendations (Priority Order)

1. **Connect pipeline to API** — Use `runPipeline()` in `/generate-video`, replace hardcoded `Root.tsx`
2. **Add persistent job store** — SQLite or JSON file for job state
3. **Add input validation + auth** — Zod schemas + simple API key
4. **Add timeout to Python exec** — `execAsync(cmd, { timeout: 120000 })`
5. **Add tests** — Vitest for TS, pytest for Python
6. **Add CI/CD** — GitHub Actions: lint → typecheck → test → build
7. **Implement Whisper alignment** — Replace estimation with real forced alignment
8. **Add logging** — Pino or Winston with levels
