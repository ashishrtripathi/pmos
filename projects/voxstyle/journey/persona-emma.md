# Customer Journey — Emma (Educator)

## Emma's Path: From Lesson Plan to Educational Video

**Persona**: Online Course Creator, 42, Intermediate
**Quote**: "I want to focus on teaching, not on video production."

---

### Journey Steps (Left → Right)

| Step | Activity | Tasks | Pain Points | Screen |
|------|----------|-------|-------------|--------|
| **1. Discovery** | Find educational video tool | Search "AI video for education", check educator forums | Needs to be accessible, reliable, affordable | Landing Page |
| **2. Sign Up** | Create educator account | Sign up, explore education-specific features | Can't find education-specific options upfront | Sign Up Form |
| **3. Choose Educational Template** | Pick lesson format | Select "Explainer" or "Tutorial" template | Templates are generic, not tailored for education | Template Gallery |
| **4. Enter Subject** | Describe the lesson | Type topic "Photosynthesis", set "Introductory" level | Worried AI will get scientific details wrong | Subject Input |
| **5. Review Script for Accuracy** | Verify content | Read script, check facts, correct terminology | No easy way to verify scientific accuracy | Script Review |
| **6. Add Accessibility** | Enable captions & clear audio | Turn on captions, select clear voice, check contrast | Accessibility is an afterthought in most tools | Accessibility Settings |
| **7. Preview & Iterate** | Review for clarity | Watch video, check pacing, ensure comprehension | Can't adjust speed or complexity per scene | Video Preview |
| **8. Export for LMS** | Upload to course platform | Download in LMS-compatible format, add to module | Different LMS have different video requirements | Export Options |
