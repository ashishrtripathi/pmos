# Customer Journey — Mike (Marketing Manager)

## Mike's Path: From Campaign Brief to Published Video

**Persona**: Small Business Marketing Manager, 35, Beginner
**Quote**: "I need to go from idea to published video in under an hour."

---

### Journey Steps (Left → Right)

| Step | Activity | Tasks | Pain Points | Screen |
|------|----------|-------|-------------|--------|
| **1. Discovery** | Research video tools | Google "easy video maker for business", compare options | No time to try multiple tools, need ROI proof | Landing Page |
| **2. Sign Up** | Create team account | Use work email, set up workspace | Need approval for paid tools, unclear free tier limits | Sign Up Form |
| **3. Create Project** | Start a video project | Name project, set brand colors/logo | No existing brand kit, have to manually configure | Project Setup |
| **4. Enter Subject** | Describe the product video | Type product name, pick "Product Demo" template | Not sure what AI can do with minimal input | Subject Input |
| **5. Generate Script** | Review and approve | Read scenes, ask team for feedback, approve | Need stakeholder sign-off before committing | Scene Table Review |
| **6. Add Branding** | Apply company identity | Upload logo, set colors, add tagline | Logo placement is guesswork, colors don't match | Brand Kit Editor |
| **7. Team Review** | Share with stakeholders | Generate preview link, collect feedback | Sharing is clunky, feedback is via email/chat | Share Preview |
| **8. Export & Distribute** | Publish to channels | Download for LinkedIn, YouTube, website | Different specs per platform, manual re-export | Export Matrix |
